# SGKit

SGKit 是一个 UI 组件合集。


## 包含组件

具体使用方式参看 demo。

- SGTransformView - 自由变换视图，适用于贴纸。
- SGPageView - 类似网易新闻。
- SGSwipeView - 类似半塘。
- SGImagePlayer - 轮播图。
- SGCollectionViewFlowLayout - 自定义 UICollectionView Section 的背景颜色。
- SGGuideView - 引导图。
